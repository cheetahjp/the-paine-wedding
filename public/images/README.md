# Wedding Site Images

Drop photos into the correct folder below. The site will automatically use them
instead of the placeholder images. Use `.webp` format for best performance.

## Folder Guide

| Folder | What goes here | Expected filenames |
|---|---|---|
| `hero/` | Full-bleed homepage background | `hero-1.webp` |
| `engagement/` | Couple photos + Open Graph image | `engagement-1.webp`, `engagement-2.webp`, `og-image.jpg` (1200×630) |
| `story/` | One photo per Our Story milestone | `story-how-we-met.webp`, `story-dating.webp`, `story-proposal.webp` |
| `bridal-party/` | Headshots for each member | `firstname-lastname.webp` (e.g. `ashlyn-bimmerle.webp`) |
| `venue/` | Davis & Grey Farms photos | `venue-exterior.webp`, `venue-interior.webp` |
| `attire/` | Style inspiration photos | `ladies-1.webp`, `ladies-2.webp`, `ladies-3.webp`, `gents-1.webp`, `gents-2.webp`, `gents-3.webp` |

## Notes

- The site falls back to Unsplash placeholder images if a file isn't found yet,
  so you can add photos gradually without breaking anything.
- For `og-image.jpg` (the social share preview), export at exactly **1200 × 630 px**.
- `.webp` is preferred — most modern tools export it. `.jpg` and `.png` also work.
- Filename casing matters — use all lowercase with hyphens (no spaces).
